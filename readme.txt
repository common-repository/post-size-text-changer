Allow your readers to change the font size of article!

== Description ==

This plugin allows your blog to change the font size of entries "on the fly".

Features:

*   Customizable HTML-wrapper to the style of your theme.
*   Automatic insertion of the plugin code all pages of posts.

== Installation ==

1. Unzip files.
2. Upload 'post-text-size-changer' to the '/wp-content/plugins/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to your 'Options'â†’'Post size text changer'.
5. Adjust the html wrapper to style plugin inserts for your theme.
